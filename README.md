# static-edge-runtime

This is a small Node.js example that starts an Express server and connects to static-edge-runtime.

## Development

Install dependencies and start:

```bash
npm install
npm start
```

The app listens on port 3002 by default and responds with "Hello World!" at the root path.

## Notes

- The package.json pins `static-edge-runtime` to version 1.4.3.
- You can change the port by setting the PORT environment variable and updating app.js accordingly.
